## 🔧 입력 방법

### 1단계: 노션 DB 열기
```
https://www.notion.so/203e5f74c72e815d8f39d2946ee85c0a
```

### 2단계: 각 업체별 주소 입력
1. **ID: 1 (골든 타이 마사지)** 행 찾기
2. **kr 업체주소** 칸 클릭 → 위의 한국어 주소 복사 붙여넣기
3. **vn 업체주소** 칸 클릭 → 위의 베트남어 주소 복사 붙여넣기
4. **ID: 2~6** 까지 반복

### 3단계: 저장 확인
- 노션에서 자동 저장됨
- 입력 후 웹사이트에서 바로 확인 가능
